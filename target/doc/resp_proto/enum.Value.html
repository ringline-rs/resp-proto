Expand description

A RESP protocol value.

This enum supports both RESP2 and RESP3 types. RESP3 types are only available when the resp3 feature is enabled.

Variants§

§

SimpleString(Bytes)

Simple string: +OK\r\n

§

Error(Bytes)

Error: -ERR message\r\n

§

Integer(i64)

Integer: :1000\r\n

§

BulkString(Bytes)

Bulk string: $6\r\nfoobar\r\n

§

Null

Null value. RESP2: $-1\r\n or *-1\r\n RESP3: _\r\n

§

Array(Vec<Value>)

Array: *2\r\n...

§

Boolean(bool)

Boolean: #t\r\n or #f\r\n

§

Double(f64)

Double-precision floating point: ,3.14159\r\n

§

BigNumber(Bytes)

Big number (arbitrary precision): (12345678901234567890\r\n Stored as the raw decimal string representation.

§

BulkError(Bytes)

Bulk error: !<len>\r\n<error>\r\n

§

VerbatimString

Verbatim string: =<len>\r\ntxt:<data>\r\n The format is a 3-byte encoding type (e.g., “txt”, “mkd”).

Fields

§format: [u8; 3]

3-byte format identifier (e.g., b“txt“, b“mkd“)

§data: Bytes

The actual data

§

Map(Vec<(Value, Value)>)

Map: %<len>\r\n<key><val>...

§

Set(Vec<Value>)

Set: ~<len>\r\n<elem>...

§

Push(Vec<Value>)

Push message (server-initiated): ><len>\r\n<elem>...

§

Attribute

Attribute (metadata attached to next value): |<len>\r\n<attrs>...<value>

Fields

§attrs: Vec<(Value, Value)>

Metadata key-value pairs

§value: Box<Value>

The actual value this metadata is attached to
